# Создание презентаций

Презентации — важная часть учёбы и работы.

## Инструменты для презентаций

| Инструмент | Где использовать |
|------------|------------------|
| PowerPoint | Для обычных докладов |
| Google Slides | Для онлайн-работы |
| Canva | Для красивых дизайнов |
| LaTeX Beamer | Для научных презентаций |

## Мои советы

1. **Не пиши много текста** — используй картинки
2. **Один слайд = одна идея**
3. **Шрифт** — крупный и читаемый
4. **Цвета** — не больше 3-4

## Что я использую

Для учёбы я использую **PowerPoint** — просто и удобно. Для научных докладов — **LaTeX Beamer**.
