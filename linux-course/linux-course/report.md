#### 1.5 Запуск исполняемых файлов

**Задание:** Научиться запускать программы из терминала.

**Выполнение:** Изучил права доступа и запуск скриптов:

1. **Права доступа (chmod):**
   - `r` (read) — чтение
   - `w` (write) — запись
   - `x` (execute) — выполнение

2. **Пример:**
   ```bash
   # Создать скрипт
   echo "echo 'Hello World'" > script.sh
   
   # Дать права на выполнение
   chmod +x script.sh
   
   # Запустить
   ./script.sh
